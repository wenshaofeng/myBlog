#### 程序翻译和程序解释

程序翻译与程序解释
◆ 翻译过程生成新的L0程序，解释过程不生成新的L0程序
◆ 计算机执行的指令都是L0
◆ 解释过程由L0编写的解释器去解释L1程序

- 常见程序翻译型语言
C/C++
Object-C
Golang
 
- 常见程序解释型语言
Python
Php
Javascript
